# create_convexhull

ポイントから凸包ポリゴンを作成するサンプル
  
  
<img src="..\..\..\..\_images\Convexhull.png" width="500">
## 使用している製品

ArcGIS 10.3.x for Desktop

## パラメータ

* 入力ポイント データ   
* 出力ポリゴン データ
* グループ フィールド指定の有無
* グループ フィールド
* バッファ作成の有無
* バッファ距離

## 内容
ポイント データから凸包ポリゴンを作成します。グループ フィールドを指定した場合、フィールド内の同一の値を持つポイントを使用して、凸包ポリゴンを作成します。また、結果にバッファを作成した状態で出力することも可能です。
