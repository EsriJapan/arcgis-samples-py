# polygon_to_line

ポリゴンの枠線をラインとして出力するサンプル
  
  
<img src="..\..\..\..\_images\PolygonToLine.png" width="500">
## 使用している製品

ArcGIS Pro 1.1

## パラメータ

* 入力ポリゴン データ   
* 出力ライン データ

## 内容
ポリゴンの枠線をラインとして出力します。ポリゴンの枠線は、Polygon.boundary プロパティから取得できます。
