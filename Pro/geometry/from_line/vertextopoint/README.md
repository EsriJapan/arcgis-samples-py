# vertextopoint

ライン データの頂点をポイント データに変換します。
  
  
<img src="..\..\..\..\_images\AddVertexPoint.png" width="500">
## 使用している製品

ArcGIS Pro 1.1

## パラメータ

* 入力ライン データ   
* 出力ポイント データ  

## 内容
入力ラインデータの頂点をポイント データに変換します。
